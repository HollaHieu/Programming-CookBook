# 01 Flask  - Request
==============================

> Implementation of vanilla Flask, using pure HTML forms and experimenting with:


- Flask Session
- Flask Request

- Lower complexity by usingHTML form manipulation (Not using WTForms)
- Some Backend tricks to change CSS status using Jinja If statements

- Experimentation of the Search Bar
- Using Session to login / Logout the user **Not meant to be implemented in a real project as is not secure**

- Using [Bootstrap4](https://getbootstrap.com/) for front-end
